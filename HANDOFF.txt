KANA TRAINER (kana.html) — CURRENT STATE  (Home menu → sections, ‹ back nav; one screen visible)
 ✅ 📖 Intro — what hiragana/katakana are, how the kana system works, how to use the app.
 ✅ 🎴 Learn & Write —
     • RECOGNIZE: flashcard; tap to flip. ORDER ALGORITHM per row of 5: phase 'forward' (A→O,
       cycles the row in order) until all 5 graded "Know it" → phase 'reverse' (O→A) until all 5
       "Know it" again → phase 'random' (weighted random across ALL introduced kana). Header shows
       the phase ("in order →/reverse ←/shuffled 🔀"). Self-grade 4 buckets (Know it/Kinda/Don't
       really/No clue). On flip: SOUND AUTO-PLAYS (ja-JP TTS), mutable via 🔊/🔇 button (persisted
       localStorage 'fkti_sound'); mnemonic TEXT+IMAGE hidden until 🖼 mnemonic tapped.
     • QUIZ GATE: finishing a row (phase random + all mastered) requires a 5-question multiple-
       choice quiz (kana→romaji) — must be perfect to unlock the next row (buildGate/renderGate/addRow).
     • WRITE: finger-tracing over a faint guide char on canvas (Clear / Hide-guide / Next).
 ✅ 🔊 Sounds & Extras — teaching + tap-to-check practicelets: dakuten ゛/handakuten ゜(voiced +
     bilabial p), yōon combos (small ゃゅょ), long vowels (おう/ー), small っ (sokuon).
 ✅ ✏️ Stroke order — KanjiVG. ▶ Watch animates strokes in order/direction (slow ~2.2s/stroke).
     Draw it yourself → Check grades each stroke for START POINT + DIRECTION + COUNT (catches シ vs ツ).
     Detection: parse SVG <path> per stroke, sample via getPointAtLength on an offscreen measure-svg,
     normalize to 0-1, compare user stroke start-dist (<0.26) + direction dot (>0.4). Prev/Next, per script.
 ✅ 📊 Progress & decks —
     • GOJŪON CHART: every kana colored by mastery: green=Know it, yellow=Kinda, orange=Don't really,
       red=No clue, grey=Not started (lvlInfo). Tap a cell to hear it. RESPONSIVE: portrait = rows by
       consonant; LANDSCAPE/DESKTOP (matchMedia orientation:landscape | min-width:680) = traditional
       vertical gojūon (vowels あいうえお down the RIGHTMOST column, consonant columns read right→left;
       placed by vowelRow()/grid). Re-renders on resize/orientationchange.
     • CUSTOM DECKS by confidence band (bandPool): 💪 Confidence boost (lvl3 only), 🙂 Little challenge
       (≥lvl2), 😤 Challenge me (≥lvl1), 🔥 Make it hurt (everything incl. unseen). Deck grades feed
       back into the chart colors.
 46 hiragana + 46 katakana, each with k(kana)/r(romaji)/m(text mnemonic) in DATA{hira:[rows],kata:[rows]}.
