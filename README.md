# FKTI Learning · 深い関係学び

Bite-sized, offline-first web apps for learning Japanese — by FKTI (Fukai Kankei, "deeply connected").

- **`index.html`** — the landing hub.
- **`kana.html`** — **Kana Trainer**: hiragana & katakana via SRS flashcards (5-at-a-time, self-graded),
  finger-tracing practice, an Intro, and a "Sounds & Extras" teaching unit (dakuten/handakuten, yōon
  combinations, long vowels, small っ) with practicelets. Mnemonic images shown on the flip.

Single-file, no build step, works offline. See **HANDOFF.txt** for full state, the asset/licensing notes,
and the TODO (katakana images re-fetch, KanjiVG stroke-order, vocabulary module).

Mnemonic art: Wikimedia Commons — hiragana **CC0**, katakana **CC BY-SA 3.0** (attributed).
